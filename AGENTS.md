## Agent skills

### Issue tracker

Issues tracked in local markdown under `.scratch/<feature>/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical triage labels use default strings (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Repo uses multi-context domain docs via root `CONTEXT-MAP.md` pointing to per-context `CONTEXT.md` files (+ ADRs). See `docs/agents/domain.md`.
